
# Comments in Python PEP8 and Flake8 

Validate code with flake8.

## Deploy

● Create virtual environment (venv)
```
python -m venv path\to\myenv
```
● Install flake8
```
pip install flake8
```

● Run flake8 main.py
```
python main.py
```
● The code was written using PEP8 and was fully validated.
